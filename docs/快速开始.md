# 快速开始指南

本指南帮助你快速上手 Personality TTS，通过命令行最快地执行和体验各种功能。

## 🚀 30 秒快速开始

### 步骤 1: 查看帮助

```bash
python3 -m src.main --help
```

### 步骤 2: 列出音频设备

```bash
python3 -m src.main --list-devices
```

### 步骤 3: 检查 API 配置

```bash
python3 -m src.main --check-asr
```

### 步骤 4: 启动文字对话模式

```bash
python3 -m src.main
```

然后选择角色，开始对话！

### 步骤 5: 启动语音对话模式

```bash
python3 -m src.main --voice --no-aec
```

戴上耳机，开始语音对话！



## 📋 常用命令速查

### 文字对话模式

| 需求 | 命令 | 说明 |
|------|------|------|
| 默认启动 | `python3 -m src.main` | 会显示角色选择菜单 |
| 自然助手 | `python3 -m src.main --role natural` | 像朋友一样聊天 |
| 学姐助手 | `python3 -m src.main --role xuejie` | 导师评价 + 学姐关怀 |
| 幽默助手 | `python3 -m src.main --role funny` | 风趣幽默 |
| 友好助手 | `python3 -m src.main --role friendly` | 友好热情 |
| 专业助手 | `python3 -m src.main --role professional` | 专业严谨 |
| 陪伴助手 | `python3 -m src.main --role companion` | 温暖陪伴 |

### 语音对话模式

| 需求 | 命令 | 说明 |
|------|------|------|
| 耳机模式（推荐） | `python3 -m src.main --voice --no-aec` | 最稳定，推荐使用 |
| 耳机 + 自然助手 | `python3 -m src.main --voice --no-aec --role natural` | 日常聊天 |
| AEC 模式 | `python3 -m src.main --voice --device-index <N>` | 需要聚合设备 |
| 指定 ASR 模型 | `python3 -m src.main --voice --no-aec --asr-model fun-asr-realtime-2025-11-07` | 切换 ASR 模型 |

### 工具命令

| 需求 | 命令 | 说明 |
|------|------|------|
| 查看帮助 | `python3 -m src.main --help` | 查看所有参数 |
| 列出音频设备 | `python3 -m src.main --list-devices` | 查看可用的麦克风 |
| 检查 ASR 鉴权 | `python3 -m src.main --check-asr` | 验证 API Key |



## ⚡ 快速切换

### 在对话中切换角色

**文字对话模式**：
```
/role natural
/role xuejie
/role funny
```

### 切换 TTS 提供商

```
/provider
```

### 清空对话历史

```
/clear
```

### 查看当前配置

```
/info
```
